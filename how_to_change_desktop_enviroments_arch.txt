Pod slovom "GNOME" myslim Gnome 3
#Vypni GNOME (GDM) po starte
sudo systemctl disable gdm.service

#Nainstaluj LXDE
sudo pacman -S lxde

#Zapni LXDE po starte
sudo systemctl enable lxdm.service
sudo reboot now

#Potom mozeme switchovat desktopove prostredia, ako len chceme
#Vypni LXDE a zapni GNOME
sudo systemctl disable lxdm.service
sudo systemctl enable gdm.service

#Vypni GNOME a zapni LXDE
sudo systemctl enable lxdm.service
sudo systemctl disable gdm.service

#jednoduche ako facka

Potom sa da GNOME odinstalovat takto:
sudo pacman -Runs gnome gnome-extra

a potom restartovat

#zdroj: https://bbs.archlinux.org/viewtopic.php?pid=693754#p693754

Odinstalovanie gnomu znizi spotrebu ramky z 500 na 200MB (hlavne gnome-extra zerie ramku)
