# Migrate user profile between Chrome and Chromium

1. Go to 

        /home/andrej/.config/chromium/Default/`
    or

        /home/andrej/.config/google-chrome/Default/

1. Copy files
    - Current Session
    - Current Tabs
    - History
    - Last Session
    - Last Tabs

1. Paste them in the `Default` directory of the other browser:

        /home/andrej/.config/google-chrome/Default/

    or

        /home/andrej/.config/chromium/Default/
